  • Oglne narzdzie optymizacji

Narzdzie oglne - tzn. e mona je zastosowa do bardzo duego zbioru problemw, a szczeglnie takich, dla ktrych trudno sformuowa precyzyjny model matematyczny oraz takich, w ktrych mamy do przeszukania bardzo du przestrze rozwiza.

Inspiracja biologiczna

Struktura chemiczna DNA

Cechy organizmw ywych s kodowane w DNA. Czsteczki DNA s zbudowane z par komplementarnych zasad poczonych wizaniami wodorowymi, otoczonymi szkieletem z piciowglowego cukru -- deoksyrybozy oraz reszty fosforanowej.

Dugo jednej pary zasad to okoo 0.34 nm. W kadej komrce czowieka znajduje si okoo 6 mld zasad. rednio okoo 1500 sekwencji zasad koduje jeden gen. Gen jest podstawow jednostk dziedzicznoci cech. Zawiera zakodowan informacj jak zbudowa okrelone biako, jak rwnie ile, gdzie i w jakich okolicznociach je zbudowa.

DNA jest zorganizowane w chromosomy -- zwinite acuchy (rozwinity acuch z jednej komrki czowieka zajby ok 2m). Komrki rnych organizmw posiadaj rne iloci chromosomw. Np. komrki czowieka posiadaj 1 par chromosomw odpowiadajcych za dziedziczenie cech pciowych (XX-kobieta, XY-facet) oraz 22 pary autosomw odpowiadajcych za dziedziczenie innych cech.

Krzyowanie

W przypadku gamet czci chromosomw mog podlega krzyowaniu. Krzyowanie pomaga zrnicowa materia genetyczny w populacji. Dziecko moe posiada inn konfiguracj genw ni jego rodzice.

Mutacja

Niezalenie w kadym z genw moe wystpi przypadkowa mutacja. Zwykle nie jest istotna, jednak czasami moe doprowadzi do zmiany okrelonych cech osobnika.

Moemy wydzieli rne rodzaje mutacji, np. usunicie czci chromosomu, zduplikowanie czsci chromosomu, odwrcenie kawaka chromosomu, albo wstawienie czci chromosomu do innego chromosomu.

Selekcja

Osobniki w populacji mog mie rne cechy, niektre podane, inne niekoniecznie. Osobniki o lepszych cechach bd czciej miay okazj do przekazania materiau genetycznego dalej. Z czasem ich kod genetyczny bdzie coraz bardziej powszechny w populacji. W ten sposb populacja zbiega do optimum.

Przypadkiem mog by chociaby yrafy, ktre w procesie ewolucji wyhodoway sobie dugie szyje umoliwiajce im zdobywanie poywienia niedostpnego dla innych zwierzt rolinoernych.

class Individual(object):
    alleles = (0,1)
    length = 100
    
    def __init__(self, chromosome=None):
        self.chromosome = chromosome or [random.choice(self.alleles) for i in xrange(self.length)]
        self.fitness = None
    
    def evaluate(self):
        self._calculate_fitness()
    
    def mutate(self):
        self._swap_gene()
    
    def crossover(self, other):
        return self._single_point_crossover(other)    

        
    def _swap_gene(self):
        allel = random.randrange(0,self.length)
        self.chromosome[allel] = random.choice(self.alleles)
    
    def _calculate_fitness(self):
        x = self.fenotype()
        self.fitness = sin(pi*x)**20 + sin(3*pi*x)**2
        
    def fenotype(self):
        return sum([self.chromosome[n]*2**n for n in xrange(self.length)]) / 2.0**self.length

    def _single_point_crossover(self, other):
        pt = random.randrange(1,self.length)
        self.chromosome[:pt], other.chromosome[:pt] = other.chromosome[:pt], self.chromosome[:pt]
        return self, other
    
    def __repr__(self):
        return '<%s fenotype=%s, fitness=%s>' % \
               (self.__class__.__name__, 
                self.fenotype(), self.fitness)

    def __cmp__(self, other):
        return cmp(other.fitness, self.fitness)
    
    def copy(self):
        twin = self.__class__(self.chromosome[:])
        twin.fitness = self.fitness
        return twin

Populacja

Wielko populacji

Im wiksza populacja, tym wiksza rnorodno genetyczna, ale i duszy czas dochodzenia do zbienoci. W przypadku prostych problemw zwykle wystarcza populacja kilku osobnikw. Dla skomplikowanych problemw wielko populacji mona liczy w setkach/tysicach.

Prawdopodobiestwo krzyowania

Krzyowanie odpowiada za mieszanie cech wrd osobnikw. Im wiksze prawdopodobiestwo krzyowania, tym wiksza szansa na uzyskanie osignicie pozytywnej synergii midzy osobnikami. Prawidowo dobrane krzyowanie zwykle dziaa dobrze przy prawdopodobiestwie krzyowania 0.5-1.0

Prawdopodobiestwo mutacji

Mutacja peni rol szumu, ma zapewni wiksz rnorodno osobnikw w populacji. Za niskie prawdopodobiestwo mutacji sprawia, e algorytm za szybko zbiega, zwykle do nieoptymalnego rozwizania. Za wysokie prawdopodobiestwo mutacji sprawia, e algorytm staje si losowy i nie osiga zbienoci. Typowe prawdopodobiestwo mutacji to 0.01-0.05.

Typ selekcji

Okrela na jakiej podstawie dobieramy osobniki, ktre maj przej do nastpnego pokolenia. Najpopularniejsze metody selekcji to metoda ruletki lub turniejowa. Kryteria selekcji mona zmienia i skalowa w trakcie ewolucji. W zalenoci od problemu, rne sposoby selekcji mog wykazywa rn wydajno.

class Population(object):
    def __init__(self, pop_size = 10, mutation_prob = 0.01, crossover_prob = 0.8):
        self.pop_size = pop_size
        self.mutation_prob = mutation_prob
        self.crossover_prob = crossover_prob
        
        self.population = [Individual() for i in xrange(pop_size)]
        for one in self.population:
            one.evaluate()
    
    def evolve(self):
        self.population.sort()
        self._crossover()
        self._mutate()

    def _mutate(self):
        for chromosome in self.population:
            for gene in chromosome.chromosome:
                if random.random() < self.mutation_prob:
                    chromosome.mutate()
        
    def _crossover(self):
        next_population = [self.best.copy()]
        while len(next_population) < self.pop_size:
            mate1 = self._select()
            if random.random() < self.crossover_prob:
                mate2 = self._select()
                offspring = mate1.crossover(mate2)
            else:
                offspring = [mate1.copy()]
            for individual in offspring:
                individual.evaluate()
                next_population.append(individual)
        self.population = next_population[:self.pop_size]
    
    def _select(self):
        """preferred selection method"""
        return self._tournament()
    
    def _tournament(self, size=5, choosebest=0.80):
        competitors = [random.choice(self.population) for i in range(size)]
        competitors.sort()
        if random.random() < choosebest:
            return competitors[0]
        else:
            return random.choice(competitors[1:])
    
    @property
    def best(self):
        """individual with best fitness score in population."""
        for one in self.population:
            one.evaluate()
        self.population.sort()
        return self.population[0]
    
    @property    
    def average(self):
        return sum([one.fitness for one in self.population])/float(self.pop_size)

import random
from math import sqrt

class Chromosome(object):
    
    def __init__(self, chromosome=None, shuff = 0):
        self.chromosome = chromosome
        if shuff == 1:
            random.shuffle(chromosome)
        self.fitness = None
    
    def evaluate(self):
        self._calculate_fitness()
    
    def mutate(self):
        self._swap_genes()
    
    def crossover(self, other):
        return self._PMX_crossover(other)    

        
    def _swap_genes(self):
        allel_one = random.randrange(0,len(self.chromosome))
        allel_two = random.randrange(0,len(self.chromosome))
        self.chromosome[allel_one], self.chromosome[allel_two] = self.chromosome[allel_two], self.chromosome[allel_one]
    
    def _calculate_fitness(self):
        self.fitness = 0
        for i in xrange(0, len(self.chromosome)):
            self.fitness += sqrt((self.chromosome[i][1] - self.chromosome[i-1][1])**2 + ((self.chromosome[i][2] - self.chromosome[i-1][2])**2))
        
    def fenotype(self):
        return [vert[0] for vert in self.chromosome]

    def _PMX_crossover(self, other):
        size = min(len(self.chromosome), len(other.chromosome))
        p1, p2 = [0]*size, [0]*size

        # Initialize the position of each indices in the individuals
        for i in xrange(size):
            p1[self.chromosome[i][0]] = i
            p2[other.chromosome[i][0]] = i
        # Choose crossover points
        cxpoint1 = random.randint(0, size)
        cxpoint2 = random.randint(0, size - 1)
        if cxpoint2 >= cxpoint1:
            cxpoint2 += 1
        else: # Swap the two cx points
            cxpoint1, cxpoint2 = cxpoint2, cxpoint1
    
        # Apply crossover between cx points
        for i in xrange(cxpoint1, cxpoint2):
            # Keep track of the selected values
            temp1 = self.chromosome[i]
            temp2 = other.chromosome[i]
        
            # Swap the matched value
            self.chromosome[i], self.chromosome[p1[temp2[0]]] = temp2, temp1
            other.chromosome[i], other.chromosome[p2[temp1[0]]] = temp1, temp2
            # Position bookkeeping
            p1[temp1[0]], p1[temp2[0]] = p1[temp2[0]], p1[temp1[0]]
            p2[temp1[0]], p2[temp2[0]] = p2[temp2[0]], p2[temp1[0]]
                    
        return self, other
    
    def __repr__(self):
        return '<%s fenotype=%s ... , fitness=%s>' % \
               (self.__class__.__name__, 
                 self.fenotype()[:5], self.fitness)

    def __cmp__(self, other):
        return cmp(self.fitness, other.fitness)
    
    def copy(self):
        twin = self.__class__(self.chromosome[:])
        twin.fitness = self.fitness
        return twin

class Population(object):
    
    def __init__(self, data_src = None, pop_size = 40, mutation_prob = 0.00, crossover_prob = 1.0):
        self.pop_size = pop_size
        self.mutation_prob = mutation_prob
        self.crossover_prob = crossover_prob
        
        self.population = [Chromosome(chromosome = data_src, shuff = True) for i in xrange(pop_size)]
        for one in self.population:
            one.evaluate()
    
    def evolve(self):
        self.population.sort()
        self._crossover()
        self._mutate()

    def _mutate(self):
        for chromosome in self.population:
            for gene in chromosome.chromosome:
                if random.random() < self.mutation_prob:
                    chromosome.mutate()
        
    def _crossover(self):
        next_population = [self.best.copy()]
        while len(next_population) < self.pop_size:
            mate1 = self._select()
            if random.random() < self.crossover_prob:
                #print "crossing over!!"

                offspring = mate1.crossover(mate2)

                offspring = [mate1.copy()]

W zapytaniach SQL znajdowao si wiele operatorw JOIN aczcych dane z tabeli znajdujcych si w rnych wzach systemu rozproszonego.

Wydajno zapyta z uyciem JOINw moe byc rna w zalenosci od tego w jakiej kolejnoci tabele bd ze sob czone. Jedn z niewtpliwych zalet algorytmw genetycznych jest dobra moliwo zrwnoleglenia. Przykadowym podejciem do zrwnoleglania AG jest uruchomienie kilku instancji AG i okresowa migracja czci osobnikw midzy pokoleniami.

Jacek Starzyski, Robert Szmuro, Bartosz Sawicki - pracownicy Wydziau Elektrycznego PW stworzyli system ECJEXT rwnolegy system optymalizacji metodami ewolucyjnymi, stworzony z myl o rozwizywaniu problemw odwrotnych elektromagnetyzmu.

Przykadowy problem - stymulacja nerwu bdnego

Celem przykadu jest optymalne zaprojektowanie magnetycznego stymulatora nerwu bdnego. Nerw bdny jest czci autonomicznego ukadu nerwowego. Nerw moe by stymulowany elektrodami umieszczonymi chirurgicznie w okolicach szyi, co pozwala zapobiega napadom padaczkowym i pomaga w walce z depresj.

Stymulacja magnetyczna pozwala unikn nacinania skry i jest znacznie bardziej bezbolesn metod ni stymulacja elektryczna.

Rwnanie rniczkowe, rozwizywane na modelu numerycznym ciaa czowieka, zdyskretyzowanym w postaci siatki czworociennej o 632808 krawdziach i 114405 wzach.

Parametry cewki

Parametry cewki: promie zewntrzny, wewntrzny, odlego od powierzchni skry, kt obrotu.

Cewka o optymalnych parametrach i rozoenie pola elektrycznego indukowanego na powierzchni skry.
